## Ветки. Локальная работа с ветками в git.

### Что такое ветки и для чего они нужны при работе с системой контроля версий.

* git branch - выводит на экран список веток

* git branch <имя ветки> - создает новую ветку

* git checkout <имя ветки> - переходит в соответствующую ветку

* git merge <имя ветки> - сливает ветку в ту, из которой вызывается эта команда

* git branch -d <имя ветки> - удаляет ветку
